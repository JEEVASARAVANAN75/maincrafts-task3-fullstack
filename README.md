# maincrafts-task3-fullstack

# ⚡ Full Stack Web Development Internship – Task 3

This project is part of the **Full Stack Web Development Internship** at Maincrafts Technology.

## 📌 Task Description

In Task 2, a multi-page website was developed.
In this task, JavaScript interactivity is added by implementing a **Contact Form with client-side validation and LocalStorage functionality**.

---

## 🎯 Objectives

* Handle form submissions using JavaScript
* Store data in browser using LocalStorage
* Dynamically display stored data

---

## 🛠️ Features

* 📄 Contact Form with:

  * Name
  * Email
  * Message
* ✅ Client-side validation (no empty fields allowed)
* 💾 Save form data using LocalStorage
* 📊 Display submitted data on a separate **Submissions Page**

---

## 📂 Project Structure

/task3
│── index.html (Contact Form)
│── submissions.html (Display Data)
│── style.css
│── script.js

---

## 🚀 How It Works

1. User fills out the contact form
2. JavaScript validates the input fields
3. Data is stored in LocalStorage
4. User can view all submissions on the submissions page
